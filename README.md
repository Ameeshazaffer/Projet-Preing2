# Projet C-WildWater – préING2

Pour compiler, dans votre terminal tapez : make

Puis voici des exemples de commandes correctes :

./myScript.sh wildwater.dat histo max
./myScript.sh wildwater.dat histo src
./myScript.sh wildwater.dat histo real
./myScript.sh wildwater.dat leaks "Facility complex #RH400057F"

Si le script n’est pas exécutable, il faut d’abord taper :

chmod +x myScript.sh
